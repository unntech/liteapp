一个PHP的轻量框架


## 主要新特性

* 采用`PHP7`强类型（严格模式）
* 支持更多的`PSR`规范
* 原生多应用支持
* 对IDE更加友好
* 统一和精简大量用法


> LiteApp 1.0的运行环境要求PHP7.0+，兼容PHP8.1

## 安装

~~~
composer create-project lzs9605/liteapp yourApp
~~~

~~~
将目录config.sample 改名为 config，可以更据需求增加配置文件
读取例子见：tests/sample.config.php
~~~

如果需要更新框架使用
~~~
composer update lzs9605/litephp
~~~
